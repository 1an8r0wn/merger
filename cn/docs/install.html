安装

我们仅建议使用已经发布的版本，而非复制整个正在开发过程中的代码仓库。

在使用任何版本以前，请确保已经阅读相应的版本说明。

在开始以前

?> 我们强烈建议通过 Watch 功能订阅版本更新（Release Only)，以便获取后续更新并及时升级。

前往 GitHub.com 订阅

!> 新版本发布后，你在 GitHub Settings 里指定的主要邮箱（Primary Email）将会收到一封提示邮件。
